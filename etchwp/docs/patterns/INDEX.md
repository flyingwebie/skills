## Sliders (Swiper.js)

Exemples de sliders issus de Swiper Studio. Chaque fichier HTML contient la structure, les styles, et le script d'init complet.

Effets natifs (pas de fichier externe) :
- `sliders/01-slide.html` : slide basique, centré, pagination
- `sliders/06-slide.html` : slide responsive (breakpoints)
- `sliders/09-slide.html` : slide basique minimal
- `sliders/11-slide.html` : slide vertical, parallax, freeMode, mousewheel
- `sliders/12-slide.html` : slide parallax, autoplay, pagination
- `sliders/13-slide.html` : slide basique minimal
- `sliders/17-slide.html` : slide navigation + pagination
- `sliders/07-coverflow.html` : coverflow, loop, autoplay
- `sliders/20-coverflow.html` : coverflow, navigation, pagination
- `sliders/14-cards.html` : cards, grabCursor
- `sliders/15-cards.html` : cards vertical, centré, pagination
- `sliders/10-creative.html` : creative, rotation 3D

Effets custom (fichiers `.min.css` + `.min.js` inclus dans le dossier) :
- `sliders/19-expo.html` + `effect-expo.min.*` : parallax + scale
- `sliders/18-super-flow.html` + `effect-super-flow.min.*` : fragments parallax
- `sliders/05-material.html` + `effect-material.min.*` : material design
- `sliders/08-carousel.html` + `effect-carousel.min.*` : carrousel 3D
- `sliders/03-panorama.html` + `effect-panorama.min.*` : panorama rotatif
- `sliders/02-tinder.html` + `effect-tinder.min.*` : swipe type Tinder
- `sliders/16-shutters.html` + `effect-shutters.min.*` : volets
- `sliders/04-cards-stack.html` + `effect-cards-stack.min.*` : pile de cartes

Chaque fichier contient le JSON indente du pattern.
Pour utiliser dans Etch, minifier le JSON avant de coller.

## FAQ Schema Markup (JSON-LD), AI-citation ready

Required fields for every `Question` entry: `dateModified` (freshness), `inLanguage`, `author`, and source citation inside the answer. Entity schema (Person / Organization / Place) goes on the page that publishes this FAQ block.

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "inLanguage": "en-IE",
  "dateModified": "{{YYYY-MM-DD}}",
  "about": {
    "@type": "Organization",
    "name": "{{CLIENT_NAME}}",
    "url": "{{CLIENT_URL}}"
  },
  "mainEntity": [
    {
      "@type": "Question",
      "name": "{{QUESTION_1}}",
      "dateModified": "{{YYYY-MM-DD}}",
      "answerCount": 1,
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "{{ANSWER_1}}",
        "dateModified": "{{YYYY-MM-DD}}",
        "author": {
          "@type": "Person",
          "name": "{{AUTHOR_NAME}}",
          "jobTitle": "{{AUTHOR_TITLE}}",
          "url": "{{AUTHOR_URL}}"
        },
        "citation": [
          { "@type": "CreativeWork", "name": "{{SOURCE_TITLE}}", "url": "{{SOURCE_URL}}" }
        ]
      }
    }
  ]
}
```

### Source Attribution Block (before JSON-LD, in the markdown FAQ)

Every answer that cites a factual claim MUST include a `fact_sources` HTML comment in the surrounding page so LLMs can trace the claim:

```markdown
<!-- fact_sources:
- claim: "Short paraphrase of the specific claim"
  source_url: "https://..."
  source_title: "Title of the cited document"
  accessed: "YYYY-MM-DD"
-->
```

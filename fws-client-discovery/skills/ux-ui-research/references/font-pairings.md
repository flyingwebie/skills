# Typography Pairings — Curated Combinations

## How to Use
Select a pairing that matches industry, brand voice, and design style. Each pairing includes Google Fonts import URL and CSS implementation.

---

## Professional / Corporate

### FP-1: "Executive Authority"
- **Heading**: Playfair Display (Serif)
- **Body**: Source Sans 3 (Sans-serif)
- **Voice**: Traditional authority, established, trustworthy
- **Best for**: Law firms, accounting, financial services
- **Import**: `https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&family=Source+Sans+3:wght@300;400;600&display=swap`

## Size Scale Recommendation

| Element | Mobile | Desktop |
|---------|--------|---------|
| H1 | 28-32px | 40-48px |
| H2 | 24-28px | 32-36px |
| H3 | 20-22px | 24-28px |
| Body | 16px | 16-18px |
| Small | 14px | 14px |
| Caption | 12px | 12px |

**Line Heights**:
- Headings: 1.2-1.3
- Body: 1.5-1.6
- Small text: 1.4

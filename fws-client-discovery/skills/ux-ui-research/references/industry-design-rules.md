# Industry Design Rules — 100 Rules Database

## How to Use
Find the client's industry category below. Each rule provides a specific design recommendation with reasoning. Rules are cumulative — start with Universal, then apply industry-specific.

---

## Universal Rules (All Industries)

1. **Mobile-first**: Design for 375px width first, then scale up. Over 60% of web traffic is mobile.
2. **Above-the-fold CTA**: Primary call-to-action visible without scrolling on all devices.
3. **F-pattern reading**: Place critical information along the F-pattern (top, left side, horizontal bars).
4. **3-second rule**: Value proposition must be understood within 3 seconds of landing.
5. **White space**: Minimum 20% white space around key content blocks for readability.
6. **Consistent spacing**: Use an 8px grid system for all spacing (8, 16, 24, 32, 48, 64).
7. **Max line length**: Body text max 65-75 characters per line for optimal readability.
8. **Visual hierarchy**: Use size, weight, color, and spacing to create clear information hierarchy.
9. **Loading speed**: Target < 3 second load time. Every second of delay = 7% conversion loss.
10. **Trust signals above fold**: Logo, phone number, or trust badges visible on first load.

## Professional Services (Legal, Accounting, Consulting)

11. **Conservative palette**: Blues, grays, deep greens. Avoid bright or playful colors.
12. **Serif headings**: Use serif fonts for headings to convey authority and tradition.
13. **Photo style**: Professional headshots, formal settings. No stock photos of handshakes.
14. **Team prominence**: Individual profiles with credentials visible from main navigation.
15. **Contact always visible**: Phone number in header, contact CTA on every page.
16. **Case results**: Showcase specific outcomes with numbers, not vague testimonials.
17. **Resource section**: White papers, guides, blog — demonstrate thought leadership.
18. **Formal tone**: Interface language should be professional, not casual.
19. **Credential display**: Certifications, memberships, awards prominently displayed.
20. **Privacy focus**: Extra emphasis on confidentiality and data security messaging.

## Healthcare / Medical

21. **Calming palette**: Soft blues, greens, whites. Avoid clinical coldness.
22. **Accessible first**: WCAG AA minimum, ideally AAA. Medical sites have legal obligations.
23. **Large text**: Minimum 18px body text. Patient demographic often needs larger type.
24. **Clear navigation**: Conditions, providers, locations, insurance as top-level nav items.
25. **Provider credentials**: Medical degrees, specializations, years of experience prominently shown.
26. **Patient portal link**: If applicable, make it prominent but don't let it dominate.
27. **HIPAA awareness**: No chat widgets that might inadvertently collect PHI without disclaimers.
28. **Appointment CTA**: "Book Appointment" must be the most prominent action on every page.
29. **Insurance info**: Accepted insurance plans easily findable, not buried.
30. **Empathetic imagery**: Real patients (with consent) or warm, diverse illustrations. No clinical stock.

## E-commerce / Retail

31. **Product-first layout**: Products visible within 1 scroll from any page.
32. **Search prominently**: Search bar large and accessible, with autocomplete.
33. **Cart always visible**: Cart icon with item count in header, persistent across pages.
34. **Product cards**: Image, title, price, rating visible without clicking. Hover for quick-add.
35. **Urgency signals**: Stock levels, limited-time pricing, "X people viewing" (use honestly).
36. **Filters that work**: Price, category, size, color, rating — sidebar or top bar, ajax-updating.
37. **Review integration**: Star ratings visible on product cards AND product pages.
38. **Trust badges**: SSL, payment icons, money-back guarantee near checkout.
39. **One-click checkout**: Minimize checkout steps. Guest checkout always available.
40. **Mobile product page**: Swipeable gallery, sticky add-to-cart button, thumb-friendly sizing.

## Technology / SaaS

41. **Modern aesthetic**: Glassmorphism, gradients, or clean flat design. Show innovation.
42. **Interactive demos**: Product screenshots should be interactive, not static.
43. **Feature sections**: Icon + headline + description pattern. Scannable, benefit-focused.
44. **Pricing clarity**: Tier comparison table with feature checkmarks. Annual/monthly toggle.
45. **Social proof bar**: Logo carousel of customers. "Trusted by X companies."
46. **Free trial CTA**: "Start Free Trial" or "Get Demo" as primary action. Low commitment.
47. **Integration showcase**: Show compatible tools/platforms with recognizable logos.
48. **Changelog signal**: Show active development. "Last updated" dates build confidence.
49. **Dark mode option**: Tech audience expects it. Implement as toggle.
50. **Developer section**: If API exists, clear documentation link. Code snippets on landing page.

## Restaurant / Food & Beverage

51. **Appetite-driving photos**: Large, high-quality food photography. No stock food photos.
52. **Warm palette**: Reds, oranges, warm browns, creams. Stimulate appetite.
53. **Menu access**: Menu accessible in max 1 click from homepage. PDF + HTML versions.
54. **Hours and location**: Visible on every page (header or footer). Not just on contact page.
55. **Reservation CTA**: "Reserve a Table" as primary action. Integrate with booking system.
56. **Mobile menu design**: Thumb-scrollable menu. Sections with sticky headers.
57. **Delivery/takeout**: If offered, order online button as prominent as dine-in.
58. **Atmosphere photos**: Interior, ambiance shots alongside food. Set expectations.
59. **Events section**: If applicable, upcoming events with easy booking.
60. **Speed over complexity**: Restaurant sites must load fast. Image optimization critical.

## Construction / Trades / Home Services

61. **Bold typography**: Strong, impactful headings. Convey reliability and strength.
62. **Portfolio gallery**: Before/after photos. Grid layout with lightbox for detail.
63. **Service area map**: Interactive or static map showing coverage area.
64. **License/insurance**: Prominently display license numbers and insurance status.
65. **Free quote CTA**: "Get a Free Quote" as primary action on every page.
66. **Project timeline**: Show typical process: consultation → quote → work → completion.
67. **Video testimonials**: Customer testimonials on video outperform text in trades.
68. **Emergency info**: If 24/7 service, make emergency contact extremely prominent.
69. **Industrial palette**: Dark blues, charcoals, oranges, yellows. Safety/industrial feel.
70. **Simple forms**: Quote forms with max 5-6 fields. Don't ask for budget upfront.

## Education / Training

71. **Organized navigation**: Course catalog, enrollment, student resources as main nav.
72. **Search by topic**: Course finder/search tool prominent on homepage.
73. **Student outcomes**: Employment rates, salary increases, student success stories.
74. **Instructor profiles**: Credentials, expertise, teaching approach.
75. **Accessible design**: Multiple learning needs. Strong WCAG compliance.
76. **Enrollment CTA**: "Enroll Now" or "Apply" with clear deadline information.
77. **Content preview**: Sample lessons, curriculum overview, syllabus downloads.
78. **Community signals**: Student forums, alumni network, events calendar.
79. **Financial info**: Tuition, financial aid, scholarship information easily accessible.
80. **Progress indicators**: If online learning, show completion tracking in UI patterns.

## Creative / Agency / Design

81. **Portfolio-first**: Work samples dominate. Case studies with process and results.
82. **Bold/experimental design**: Push boundaries. The website IS the portfolio.
83. **Animation tasteful**: Subtle scroll animations, hover effects. Don't overdo.
84. **Dark mode default**: Creative agencies can pull off dark backgrounds effectively.
85. **Process showcase**: Visual representation of how you work. Timeline or steps.
86. **Client logos**: Display recognizable clients. Social proof through association.
87. **Culture/team**: Show personality. Behind-the-scenes, team culture content.
88. **Awards/recognition**: Industry awards, design competitions displayed prominently.
89. **Minimal text**: Let the work speak. Copy should be sharp and concise.
90. **Contact personality**: Contact page should reflect brand personality, not be generic.

## Finance / Insurance

91. **Security-first design**: Lock icons, SSL badges, security certifications prominent.
92. **Conservative palette**: Navy, dark blue, green (money), gold (premium).
93. **Calculator tools**: Interactive tools (mortgage calc, insurance quote) drive engagement.
94. **Regulatory compliance**: Disclaimers, disclosures properly placed. Legal review.
95. **Comparison tables**: Plan/product comparison tables for decision-making.
96. **Customer portal**: Prominent login for existing customers. Don't dominate new visitor flow.
97. **Advisor profiles**: Credentials, certifications (CFP, CFA, etc.) prominently shown.
98. **Simple language**: Avoid jargon. Financial literacy varies. Plain language wins.
99. **Mobile banking**: If applicable, app download CTAs on mobile devices.
100. **Data visualization**: Use charts and graphs to make financial data digestible.
